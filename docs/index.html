Course Description

This course covers the methods and techniques that non-profit organizations and public agencies can use to monitor and evaluate the effectiveness of their programs. It focuses on the design of primary data collection and analysis practices that provide credible evidence of a program’s impacts. Topics covered in the course include impact evaluation, performance evaluation, measurement, multi-method evaluation techniques, and the analysis of experimental and non-experimental monitoring data.

Learning Goals

  • Explain the role of monitoring and evaluation in evidence-based environmental policy and program design.

  • Develop and assess theories of change linking program inputs, outputs, outcomes, and impacts.

  • Design credible monitoring and evaluation strategies, including appropriate indicators, data collection plans, and counterfactuals.

  • Implement core monitoring and evaluation tasks in R, including data management, measurement construction, and basic impact analysis.
